# 단타 보조 시스템 — 실행 & 투자 활용 가이드

> ⚠️ **먼저 읽어주세요.** 이 프로그램은 **시세 조회·기록·학습 보조 도구**입니다.
> 자동으로 주문을 내거나 돈을 옮기지 않습니다. 실제 매수·매도는 영웅문(HTS)에서 직접 하셔야 합니다.
> 단타 매매는 **고위험**입니다. 손실이 날 수 있고, 이 문서·앱의 내용은 **투자 권유가 아니며 모든 판단과 책임은 본인**에게 있습니다.

---

## A. 실행 방법

### 1단계 — 데모로 먼저 충분히 연습 (지금 상태, 돈 안 듦)
- `시작_데모.bat` 더블클릭 → 브라우저에서 `localhost:8800`
- 샘플 데이터로 4개 패널 + 체크리스트 + 노하우를 **마음껏 연습**
- 실제 시세·실제 주문과 무관 (연습용)

### 2단계 — 실데이터로 전환 (실전 시세 보기)
실제 실시간 시세로 쓰려면 한국투자증권 API 키가 필요합니다.

1. **키움이 아니라 한국투자증권** Open API 신청
   - https://apiportal.koreainvestment.com 접속 → 한투 계좌 개설 → "KIS Developers" API 신청
   - **앱키(APP KEY)**, **시크릿(APP SECRET)**, **계좌번호** 발급
2. 폴더의 `.env.example` 파일을 복사해서 이름을 **`.env`** 로 바꾸기
3. `.env` 를 메모장으로 열어 발급받은 값 입력:
   ```
   KIS_APP_KEY=발급받은_앱키
   KIS_APP_SECRET=발급받은_시크릿
   KIS_ACCOUNT_NO=50123456-01
   KIS_MOCK=true      ← 처음엔 true(모의투자 서버)로 안전하게 테스트 권장
   ```
4. `시작_데모.bat` 다시 실행 → 이제 **🟠 DEMO 배지가 사라지고 실시간 시세**로 동작
5. 익숙해지면 `KIS_MOCK=false` 로 바꿔 실서버 시세 사용

> 💡 **중요**: 이 앱은 "보는 것"과 "기록"만 합니다. **실제 주문 클릭은 영웅문에서** 하세요.
> 앱과 영웅문을 두 화면에 띄워두고, 앱으로 판단 → 영웅문으로 주문하는 방식입니다.

## B. 하루 매매 흐름에 맞춘 사용법 (강의 "하루흐름" 기반)

### 🌅 장 전 (08:40 ~ 09:00) — 준비
- **급등 랭킹** 패널로 어제·시간외에서 강했던 종목, 오늘 주도 테마 파악
- 영웅문 "예상체결등락"과 함께 오늘의 후보를 추림
- 후보를 **관심종목** 패널에 등록(종목코드·종목명·**목표가·손절가**)

### ☀️ 장 중 (09:00 ~ 15:20) — 발굴·판단·기록
1. **급등 랭킹**에서 +20% 이상, 거래량비 높은 종목 클릭 (20%↑는 빨간 강조)
2. 클릭하면 **상한가 체크리스트**에 자동 입력 → "분석"
   - 호재 뉴스·실검 질문에 답 → **점수 70↑ "진입 검토" / 50~69 "관망" / 50↓ "제외"**
3. 진입을 결정하면 → **영웅문(HTS)에서 실제 매수 주문**
4. 매수 직후 **강일지**에 즉시 기록: `+매수` → 종목·금액·**진입 이유(호재 뉴스)**
5. **관심종목** 패널이 실시간으로 목표가/손절가 도달 시 **알림**

### 🌙 장 마감·시간외 (15:20 ~ 16:00)
- 보유 종목 매도 시 강일지에 `+매도` 기록 → **손익 자동 계산**
- 시간외 매매는 **💡 노하우 → "④ 시간외 거래"** 항목 참고
  (우선주-본주 연동, 물량 두께, 빨강/파랑 읽기, 평단 종가 인식 등)

### 📓 장 후 — 복기
- **강일지**로 오늘 매매 복기 (손실 건도 반드시 기록 → 패턴 개선)
- **📚 커리큘럼**으로 학습 단계 체크
- 다음날 관심종목 갱신

## C. 💡 노하우를 실전에서 쓰는 법

헤더의 **💡 노하우** 버튼 → 9개 분야 41개 항목 (검색 가능). 상황별로 이렇게 활용:

| 이런 순간에 | 이 분야를 보세요 |
|---|---|
| 어떤 종목을 살지 고를 때 | ① 종목 선정 (대장·부대장·상대적 세기) |
| 테마가 돌 때 | ② 순환매 & 테마 흐름 |
| 상한가 근처에서 진입 고민 | ③ 상한가 매매 & 진입 타이밍 |
| 시간외에 매매할 때 | ④ 시간외 거래 (우선주·물량) |
| **금요일이거나** 상한가가 흔들릴 때 | ⑤ 상한가 풀림 & 금요일 리스크 |
| 비중·미수 고민 | ⑥ 자금 관리 |
| 호가·화면이 헷갈릴 때 | ⑦ HTS 세팅 & 화면 |
| 실적·뉴스가 의심스러울 때 | ⑧ 재무·뉴스 검증 |
| 손절을 못 하겠을 때 | ⑨ 멘탈 & 원칙 |

> **금요일 자동 경고**: 금요일에 앱을 켜면 헤더 아래 빨간 띠로 "상한가 풀림 잦음·포지션 경량화" 경고가 자동 표시됩니다.

## D. 꼭 지킬 것 (리스크 관리)

- **소액·모의투자로 충분히 연습**한 뒤 실전 (강의도 50만~100만 원 시뮬레이션 후 단계적 확대 권장)
- 체크리스트 점수는 **참고 보조**일 뿐, 맹신 금지
- **손절 원칙**을 미리 정하고 지키기 (노하우 ⑤·⑨)
- 단타는 잃을 수 있는 돈으로만. 빚(미수·신용) 과다 사용 주의 (노하우 ⑥)
- 이 도구·문서는 교육·보조용이며 **투자 권유가 아닙니다. 최종 판단과 책임은 본인**에게 있습니다.

---

*실행이 막히거나(키 발급·.env 설정 등) 특정 화면에서 헷갈리면, 어느 단계에서 무엇이 보이는지 알려주시면 바로 도와드리겠습니다.*
