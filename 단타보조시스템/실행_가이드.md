# 단타 보조 시스템 — 실행 가이드

KIS Open API 연동 실시간 급등 랭킹·상한가 체크리스트·강일지·관심종목 모니터링 로컬 웹앱.

## 검증 상태 (2026-06-16)

- 구현: 계획서 13개 태스크 + 확장 기능(노하우·자금관리·녹취록 검색·강일지 통계/CSV·체크리스트 적중률·시장 분위기·데모 모드) 반영
- 백엔드 엔드포인트 18개, 단일 HTML 프론트(4패널 + 9개 모달/도구)
- 테스트: `pytest` **12개 전부 통과** (database 3 · kis_client 3 · api 6)
- E2E 스모크: GET/POST 엔드포인트 13종 전부 200, 9개 모달·헤더 버튼 5종·분위기바·금요일배너·CSV·결과기록 UI 동작 확인. KIS 키 미설정 시 데모 모드 자동 기동(크래시 없음).

## 1. 설치 (최초 1회)

```bash
cd 단타보조시스템
pip install -r requirements.txt
```

## 2. KIS API 키 설정

`.env.example`을 `.env`로 복사 후 키 입력:

```
KIS_APP_KEY=발급받은_앱키
KIS_APP_SECRET=발급받은_시크릿
KIS_ACCOUNT_NO=50123456-01
KIS_MOCK=false          # true 면 모의투자 서버 사용
```

키 발급: https://apiportal.koreainvestment.com

## 3. 실행

```bash
python main.py
```

브라우저에서 http://localhost:8000 접속. 헤더 배지가 "● 연결됨"이면 정상.

## 4. 화면 구성 (4패널)

| 패널 | 기능 |
|------|------|
| 🔥 급등 랭킹 | 코스피/코스닥/전체 등락률 상위 30종목, 5초 폴링(장중 09:00~15:30), 20%+ 강조 |
| ✅ 상한가 체크리스트 | 종목 클릭→자동입력, 6항목 점수화(호재30·거래량20·등락률20·실검15·시총10·연속상한가아님5), 70+ 진입검토 |
| 📓 강일지 | 매수/매도 기록, LIFO 손익 자동계산, 최근 7일 미니 바차트, SQLite 영구저장 |
| 👁️ 관심종목 | 최대 20종목 WebSocket 실시간 체결가, 목표가/손절가 도달 시 브라우저 알림 |

## 참고

- 프론트엔드는 외부 CDN 없는 단일 HTML — 오프라인에서도 UI 로드됨(데이터만 서버 필요).
- 장 마감 시간엔 랭킹 폴링이 자동 중단되어 API 호출을 아낍니다.
- 강일지·관심종목 데이터는 `trades.db`(SQLite)에 보존됩니다.

### 테스트 실행 시 주의 (샌드박스/프록시 환경)
일부 환경에서 SOCKS 프록시 env(`ALL_PROXY` 등)가 설정돼 있으면 `httpx` 생성자가 `socksio`를 요구해 `kis_client` 테스트가 환경 오류로 실패할 수 있습니다(코드 문제 아님). 이때는 프록시 env를 비우고 실행:

```bash
env -u ALL_PROXY -u HTTP_PROXY -u HTTPS_PROXY python -m pytest tests/ -v
```

일반 PC(Windows)에선 해당 env가 없어 그대로 `pytest tests/ -v`로 12개 전부 통과합니다.


## 🟠 데모(목업) 모드 — KIS 키 없이 바로 체험

`.env`에 KIS 키를 설정하지 않고 `python main.py`로 실행하면 **자동으로 데모 모드**로 기동됨.

- 헤더에 `🟠 DEMO · 샘플 데이터` 배지 표시
- 급등 랭킹: 샘플 테마주(에코프로비엠·한화오션 등) 16종목이 5초마다 실시간처럼 갱신
- 체크리스트: 샘플 현재가로 점수·판정 즉시 산출
- 관심종목: 등록 시 샘플 체결가가 실시간 업데이트(목표가/손절가 알림 체험 가능)
- 강일지·관심종목: 첫 기동 시 샘플 데이터 자동 시드 → 4개 패널 모두 즉시 채워짐(에코프로비엠·한화오션)
- 교육 커리큘럼: 실제 기능 그대로 동작

> 데모 데이터는 `demo_data.py`의 샘플이며 실제 시세가 아님.
> 데모 기록은 **별도 `demo_trades.db`**에 저장돼 실데이터(`trades.db`)와 섞이지 않음.
> KIS 키를 설정하면 자동으로 실데이터 모드(`trades.db`)로 전환됨(데모 코드 경로와 완전 분리).
